# Desafio Angular

Projeto desenvolvido como desafio técnico utilizando Angular e Angular Material.  
A aplicação consome a API pública JSONPlaceholder para exibir posts e usuários.

## Funcionalidades

- Listar posts
- Buscar posts pelo título
- Ver detalhes de um post
- Listar usuários
- Buscar usuários
- Loading enquanto os dados carregam

## Tecnologias utilizadas

- Angular
- Angular Material
- TypeScript
- HTML
- SCSS

## API utilizada

Os dados são consumidos da API:

https://jsonplaceholder.typicode.com

Endpoints usados:

- /posts
- /posts/:id
- /users

## Como executar o projeto

1. Clone o repositório
git clone https://github.com/maxweees/desafio-angular/

2. Entre na pasta do projeto

cd desafio-angular

3. Instale as dependências

npm install

4. Execute o projeto

ng serve

5. Abra no navegador

http://localhost:4200
